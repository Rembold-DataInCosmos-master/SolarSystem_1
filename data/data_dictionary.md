
# Data Dictionary

## Jupiter

| Column  | Units     | Description                                       |
|---------|-----------|---------------------------------------------------|
| Date    | timestamp | The time the position was recorded                |
| JD      | days      | Julian Date: an alternative to timestamps         |
| RA      | HMS       | Right Ascension in Hour Minute Second format      |
| DEC     | deg       | Declination in deg, arc-minute, arc-second format |
| Dist_AU | AU        | The distance from Earth to Jupiter                |
